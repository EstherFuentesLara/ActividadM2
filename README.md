# Predicción de enfermedad cardiovascular con Machine Learning ActividadM2
Practica de Machine Learning - Predicción de riesgo cardiovascular con dataset Framingham

Práctica de aprendizaje automático utilizando el dataset Framingham.

## Objetivo
Desarrollar un modelo de clasificación capaz de predecir el riesgo de enfermedad cardiovascular a 10 años.

## Tecnologías utilizadas

- Python
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn

## Modelo utilizado

Regresión logística.
